The Speeb Publisher Worpress Plugin allows you to monetize your blog by publishing sponsored posts. To use it, register at Speeb (www.speeb.com).

== Installation ==

1) Extract the zip file and drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page;

2) Once the plugin is activated, go to your Settings page at Speeb.com and copy your API Key;

3) Enter you API Key and the credentials of a valid Wordpress user with permissions to publish on your blog in the Speeb Publisher Settings
